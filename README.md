# Automation Ideas - UNI ASSIGNMENT 1
This repository contains ideas and simple approaches for improving workflows using automation and data.
## Idea 1: Automating Data Cleaning
Many datasets require repeated cleaning (removing duplicates, handling missing values).  
This process can be automated using Python scripts to save time and improve consistency.
## Idea 2: Automated Reporting
Instead of manually updating Excel reports, a script or tool can automatically update data and generate summaries or charts.
## Idea 3: Email or Notification Automation
Tasks like sending updates or reminders can be automated using simple workflows, reducing manual effort.
## Idea 4: Dashboard Simplification
Complex data outputs can be transformed into simple dashboards to make them easier for non-technical users to understand.
## Goal
To explore how automation and simple tools can reduce manual work and improve efficiency in real-world tasks.
